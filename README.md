# Mini E-Commerce Storefront

This project is a **Mini E-Commerce Storefront** developed as **Task 2** of the  
**Full Stack Web Development Internship** at **Future Interns**.

The application demonstrates core e-commerce functionality including product
listing, search, shopping cart management, and a checkout simulation.

---

## Features

- Product listing with search functionality
- Shopping cart with quantity control
- Add and remove items from cart
- Checkout simulation with form validation
- Responsive and clean user interface
- State management using React Context API


## Tech Stack

- **Frontend:** React (Vite)
- **Styling:** Tailwind CSS
- **State Management:** Context API
- **Backend:** Not required (checkout simulation)
